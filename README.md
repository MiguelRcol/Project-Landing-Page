# Project-Landing-Page
Odin project landing page repository
Landing Page (The Odin Project)

This project is a responsive landing page built as part of The Odin Project Foundations curriculum. The goal was to recreate a provided design mockup using HTML + CSS (Flexbox), focusing on layout, spacing, typography, and section structure.

What I practiced

Building a full page layout from a mockup

Flexbox for:

header navigation alignment

hero split layout (text + image placeholder)

four-card information section

call-to-action section alignment

Reusable layout pattern using a centered .container (consistent left/right edges)

Typography and spacing:

headline hierarchy (48px hero title, 36px section title)

consistent padding/margins across sections

Clean structure and semantic sections:

Hero / Info / Quote / CTA / Footer

Sections included

Header + Hero: logo, navigation links, hero text and sign-up button, image placeholder

Information: 4 cards with illustration placeholders and subtext

Quote: testimonial/quote block with author alignment

Call to Action: highlighted CTA box with text and button

Footer

Built with

HTML5

CSS3

Flexbox

How to run locally

Clone the repository

Open index.html in your browser

Notes

This project follows the design guidelines provided by The Odin Project and focuses on layout accuracy and clean CSS structure.
